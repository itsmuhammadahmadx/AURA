# AURA - Premium AI Assistant

## Setup Instructions

### Step 1 - Install dependencies
Open terminal in this folder and run:
```
npm install
```

### Step 2 - Add your API key
Open the `.env` file and replace `your_gemini_api_key_here` with your actual Gemini API key:
```
VITE_GEMINI_API_KEY=paste_your_key_here
```

### Step 3 - Run the app
```
npm run dev
```

Then open your browser and go to: http://localhost:5173

## Features
- Splash screen with glow animation
- Business onboarding (per-client setup)
- Left sidebar with chat history
- Chats saved to localStorage
- Gemini AI powered responses
- Animated dark premium UI
- Mobile responsive
